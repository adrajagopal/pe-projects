1. Start by creating a new theme folder and naming it something unique
  2. Go into your admin panel and create a couple of test pages - Home, list, details, etc
  3. Create an index.php file, and add two php functions - wp_head() and wp_footer()
  4. then, populate your index file with body content
  5. create a page router
  6. basic structure: if ( is_page() ) then include('page_name');
  7. you could also just create a variable for the page name based on the file path, and skip the conditional logic
  8. use if ( is_singular('post_plural') ) then include ( 'details.php') to display details page for an individual post from a post type
  9. use is_404() to create a 404 page
  10. create your css files - reset, setup, structure, typography, etc
